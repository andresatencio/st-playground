# ST Playground

Playground basado en el [ST Select Transform](https://github.com/selecttransform) de la librería ST.

## Cambios principales

- **Eliminación de Cell.js**: Se removió la dependencia de [Cell.js](https://www.celljs.org/) como librería para manejo de componentes.
- **Vanilla JS**: Ahora los componentes se manejan utilizando JavaScript puro (Vanilla JS).
- **Cloudflare CDN**: Las librerías requeridas para el funcionamiento del playground se obtienen mediante [Cloudflare CDN](https://cdnjs.cloudflare.com/).

## Uso

Simplemente abre el archivo `index.html` en tu navegador para comenzar a usar el playground.
O [https://andresatencio.github.io/st-playground](https://andresatencio.github.io/st-playground)
